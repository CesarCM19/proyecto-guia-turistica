# Créditos y Atribuciones de Recursos Multimedia

Este proyecto es una Guía Turística Multimedia de Costa Rica desarrollada con fines educativos para el curso **IF7102 – Multimedios**. A continuación, se detallan las fuentes, licencias y la lista completa de los recursos multimedia utilizados en la aplicación, garantizando el cumplimiento de los lineamientos de derechos de autor e integridad académica.

## Código y Herramientas

El proyecto ha sido desarrollado utilizando exclusivamente estándares de la plataforma web nativa:
* **Estructura y Lógica:** HTML5, CSS3, JavaScript ES6+ (ES Modules) y Web Components nativos (Custom Elements, Shadow DOM, HTML Templates).
* **Fuentes y Estilo:** Fuentes del sistema (`system-ui, -apple-system, sans-serif`) para optimizar el rendimiento.
* **Iconografía:** Los iconos SVG integrados provienen de conjuntos de uso libre y abierto como Material Design Icons.
* **Automatización de Audios:** Script [generar-audioguias.js](generar-audioguias.js) utilizando la biblioteca Node.js `msedge-tts`.
